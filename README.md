# Employee Tracker
This applications allows the user to track 
employee information, employee roles, and departments as well as allowing the user to delete these values.

## Possible Options
- View all departments.
- Add new departments.
- Delete departments.
- View all employees.
- Add new employees.
- Delete employees.
- View roles.
- Add new roles.
- Update roles.
- Delete roles.
